# ALMa typography

The brand uses a scholarly editorial pairing.

## Brand / headings

**Merriweather Semibold**

```css
font-family: Merriweather, Georgia, "Times New Roman", serif;
font-weight: 600;
```

Use for:

- ALMa wordmark
- marketing headlines
- editorial page titles
- hero moments

## UI / body

**Inter Regular / Medium / Semibold**

```css
font-family: Inter, ui-sans-serif, system-ui, -apple-system, BlinkMacSystemFont, "Segoe UI", sans-serif;
```

Use for:

- app navigation
- buttons
- body copy
- metadata
- labels
- form inputs

## Suggested scale

| Token | Size | Line height | Use |
|---|---:|---:|---|
| `display` | 56px | 1.05 | Landing hero |
| `h1` | 40px | 1.12 | Page title |
| `h2` | 28px | 1.2 | Section title |
| `h3` | 20px | 1.3 | Card title |
| `body` | 16px | 1.55 | Reading text |
| `small` | 14px | 1.45 | Metadata |
| `caption` | 12px | 1.4 | Tags and timestamps |

Do not embed font files in the repository unless licensing/distribution is confirmed.
