# ALMa color palette

Source of truth for the corrected ALMa frontend brand system.

| Token | Hex | Use |
|---|---:|---|
| `navy` | `#0F1E36` | Primary brand color, app icon background, headings, sidebar |
| `ink` | `#09162A` | High-emphasis text and outlines |
| `teal` | `#0E6F6D` | Accent color, links, active states, bookmark detail |
| `pale-blue` | `#B7D2E4` | Secondary surfaces, info highlights, soft cards |
| `parchment` | `#F3EAD6` | Warm neutral, library/paper accents |
| `gold` | `#C49A45` | Fine dividers, book trim, premium accents |
| `paper` | `#FFFCF7` | Main app background |
| `cream` | `#FFF9F0` | Elevated panels and cards |

## Semantic mapping

```css
--alma-bg: #FFFCF7;
--alma-surface: #FFF9F0;
--alma-text: #09162A;
--alma-heading: #0F1E36;
--alma-accent: #0E6F6D;
--alma-border: #D9CBAF;
--alma-focus: #0E6F6D;
```

## Contrast notes

- Use navy or ink text on paper, cream, parchment, and pale blue.
- Use white or cream text/icons on navy backgrounds.
- Use teal primarily as an accent, not for long body text.
- Use gold for trim and separators, not body text.
